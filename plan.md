
create the function **notebook.add_table()**

that will recibe a table in these two different formmat: 

```bash 
local itens = {
    {"name", "age", "city"},
    {"mateus", "22", "rio de janeiro"},
    {"joao", "23", "sao paulo"},
    {"maria", "24", "belo horizonte"},
}
```

or in the format 

```bash 
local itens = {
    {name = "mateus", age = "22", city = "rio de janeiro"},
    {name = "joao", age = "23", city = "sao paulo"},
    {name = "maria", age = "24", city = "belo horizonte"},
}
```

and will create a table in html in the content, containing all the data
